# Employee Management System (EMS)

This is a full-stack Employee Management System developed using Spring Boot and React.js. 
The application allows users to manage employee records with secure authentication and file upload functionality.

## 🔧 Technologies Used
- Backend: Spring Boot, Spring Security, JWT, REST API
- Frontend: React.js, Axios, React Hook Form
- Database: MySQL
- Build Tool: Maven
- Environment: Linux (WSL)

## ✨ Features
- User Authentication with JWT
- Add, Update, Delete Employees
- View Employee List
- Upload Profile Photo and Resume
- RESTful API Integration
- Secure API endpoints

## 📂 Project Structure
- ems-backend (Spring Boot API)
- ems-ui (React Frontend)

## 🚀 Future Enhancements
- Role-based access control
- Pagination & Search
- Dashboard analytics# Employee Management System (EMS)
